You can modify the behavior of Intellij IDEA using a mechanism similar to videogame patching. This guide shows how to change the startup splash screen.

  1. Execute jar tf path/to/idea/lib/resources.jar | less and search for "logo.png". Note the file name.
  2. Copy the png image you want in some directory with the name you found above. (you can pick one from here)
  3. Open the idea launcher script (idea.sh) in an editor
  4. Append this directory to the front of classpath: CLASSPATH="$PATCH_DIRECTORY:$IDE_HOME/lib/bootstrap.jar"
  5. Chown this directory to root:root and chmod to 755

Now the chosen splash screen will be shown on startup. This method can be used to override classfiles as well:

  1. Note down the build id of your installation (grep buildNumber $IDE_ROOT/product-info.json)
  2. Goto intellij-community github repo, switch to the branch for your build id, and download the class source file you wish to change.
  3. Make the changes you wish to make to that class.
  4. Compile the file and put the output in your patch directory (javac -cp $IDE_ROOT/lib/\* ModdedClass.java -d $PATCH_DIRECTORY)

Intellij is a huge project (multi million lines of code), so compiling it is a no go if you don't have a beefy machine. This is a patching mechanism, not a plugin, so you can change the ide in any way you want, however the changes won't be portable.
